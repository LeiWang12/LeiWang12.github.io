搭建Hexo博客

搭建前提

1、使用github创建一个UserName.github.io版本库文件，UserName是你的github登录名。

2、安装node.js

3、安装git

安装步骤

一、安装hexo

  1. cd /
  2. npm install hexo-cli -g

在本地硬盘新建一个空目录（eg:blog）,进入文件夹，鼠标右键打开git bash shell

安装博客所需文件

  1. cd /
  2. hexo init Hexo
  3. cd /Hexo
  4. npm install
  5. hexo generate（可简写为hexo g）
  6. hexo sever（可简写为hexo s）

如果你在本地访问http://localhost:4000/，成功看到博客网页，证明本地安装成功

部署到github

1、修改hexo博客下_config.yml文件

deploy:

type:git
repository:你的github的仓库git地址
branch:master

2、在git shell输入命令

hexo g  编译文件

hexo s   启动本地服务，检查是否正确

hexo d   上传到github服务器

坑：在github上新建的仓库名字一定要和自己登录github网站的用户名一致！！！
